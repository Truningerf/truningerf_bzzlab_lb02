My Person

Hello, my name is Fabian. I am from Switzerland and I am 16 years old. In my free time I love to build MOC's from my Lego collection. In addition, I like to take pictures, I also play video games from time to time and read a lot.


In the summer of 2020, I started my apprenticeship as a media technician. This apprenticeship is very varied and covers many different areas. For example, I'm learning how to take photos properly or create a film. I'm also learning how to fully prepare, create and manage a web presence.

About this website

I created this website as part of a project work of module 287. In this module we learned how to design and layout websites using a CSS framework. I chose Bootstrap as CSS framework because it is very easy to understand and works a lot with Responsive Design.


In April 2021, I started this project work with great pleasure, with the ulterior motive that I may need this website again later. So I wanted to make a website about me and my work at home and at school.
